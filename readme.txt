Some Basic tricks to manage admin bar, post autosave and post revision.

== Description ==

This plugin allow you to manage admin bar to show or hide in front end, change the post autosave interval or disable and limit your post revisions or disable the feature. 

Read more link http://www.tricksofit.com/2014/09/some-useful-basic-tricks-of-wordpress

= Features =

 - Hide admin bar for all users
 - Show admin bar for only admin
 - Remove wordpress generator meta tag
 - Disable post autosave or change autosave interval
 - Change limit for post revisions or disable post revision

== Installation ==

unzip archive to wp-content/plugins directory, and activate it in Plugins menu in wp-admin
Then go to settings->WP Basic Tricks and change the settings according to you.

== Frequently Asked Questions ==

= how to use this plugin =

Go to Wordpress settings -> WP Basic Tricks and change the settings according to you. If it generates a code for config.php then just copy and paste it to your config.php file.
