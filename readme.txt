== Description ==

Some Basic tricks to manage admin bar, post autosave and revisions. Read more link
http://www.tricksofit.com/2014/09/some-useful-basic-tricks-of-wordpress


== Installation ==

unzip archive to wp-content/plugins directory, and activate it in Plugins menu in wp-admin
Then go to settings->WP Basic Tricks and change the settings according to you.

== Frequently Asked Questions ==

= how to use this plugin =

Go to Wordpress settings -> WP Basic Tricks and change the settings according to you. If it generates a code for config.php then just copy and paste it to your config.php file.
